# Sandpile

一个WEB全栈试验场，基于个人需求制作

[查看页面](https://sandpile.shop/)

## 系统：

- ✅权限管理
- ✅用户登录
- 用户注册（暂不开放注册）
- ✅自动登陆功能

## Markdown笔记模块

- ✅新增、查询、重命名、删除笔记项目
- ✅解析器：解析Markdown为HTML，并生成菜单
- ✅列举笔记项目内部目录，查看笔记
- ✅编辑markdown笔记、保存编辑内容
- ✅新增、重命名、删除笔记

## 3D模块

- ✅DEMO1：以图片为模板的平面粒子
